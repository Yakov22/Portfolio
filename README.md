# Portfolio
Данный репозиторий был создан для демонстрации навыков вёрстки и написания кода на языках HTML, CSS и JS.


Портфолио представляет из себя репозиторий с исходными файлами и 
[стартовую страницу](https://www.example.com) размещённую на GitHub Pages, через которую можно перейти на проекты.  

Для сборки проектов применялся gulp:
- минификация HTML
- компиляция препроцессоров SASS
- минификация CSS  
- объединение нескольких файлов CSS(SCSS) в один
- автоматическое добавление префиксов CSS
- преобразования кода ECMAScript 2015 + в обратно совместимую версию JavaScript с помощью Babel
- минификация JavaScript
- объединение нескольких файлов JavaScript в один
- сжатие изображений

В папках dist находятся файлы обработанные Gulp - минифицированные файлы и сжатый графический контент.   
в папках src файлы исходников для ознакомления.
